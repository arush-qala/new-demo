# Models Directory

## Place Your Mixamo Model Here

1. Download realistic female model from Mixamo (GLB format)
2. Place the GLB file in this folder
3. Update `MIXAMO_MODEL_FILENAME` in `src/components/Three/ModelAvatar.tsx`
4. The model will automatically load on all product pages!

## Example:
- Filename: `realistic-woman.glb`
- Full path: `/public/models/realistic-woman.glb`
- Update code: `const MIXAMO_MODEL_FILENAME = 'realistic-woman.glb'`
