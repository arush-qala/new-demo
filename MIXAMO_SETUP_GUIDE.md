# Mixamo Setup Guide - Quick Instructions

## ✅ Download Format: **GLB** (Binary GLTF)

**This is the format you need for React Three Fiber.**

## Step-by-Step:

### 1. Go to Mixamo
- Visit: https://www.mixamo.com
- Sign in (free Adobe account)

### 2. Find Female Model
- Click "Characters" → Search "female"
- Select a realistic female model (e.g., "Ava", "Alicia", "Sofia")

### 3. Download Settings
- **Format:** Select **"GLB"** (if available) or **"FBX"** (convert later)
- **Pose:** T-Pose
- **Skin:** With Skin
- Click Download

### 4. Convert if Needed (FBX → GLB)
If you downloaded FBX:
- Use online converter: https://products.aspose.app/3d/conversion/fbx-to-gltf
- Or use Blender (free)

### 5. Place in Project
1. Create folder: `/public/models/`
2. Place your GLB file there
3. Name it something simple: `realistic-woman.glb`

### 6. Tell Me the Filename!
Once you have the file, just tell me:
- **"The filename is: realistic-woman.glb"**
- I'll update the code immediately!
